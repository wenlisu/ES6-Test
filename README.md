# activities
**templates放html、app里面放script、style、img**

**根目录下install the deps**
```bash
$ npm install && bower install
```

**run the dev server**
```bash
$ npm run start
```

then open your http://localhost:8080/index.html 

**创建新html、js、scss，img文件夹**
```
$ node --harmony tools html <DATENAME>
```
